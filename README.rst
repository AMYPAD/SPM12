SPM12
=====

|Versions| |Py-Versions| |Tests| |LICENCE|

Thin Python wrappers around `Statistical Parametric Mapping <https://www.fil.ion.ucl.ac.uk/spm>`_.

|PET-MR Coregistration|


Install
-------

.. code:: sh

    python -m pip install -U spm12
    python -m spm12
